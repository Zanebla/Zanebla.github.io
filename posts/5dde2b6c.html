电影艺术文章管理系统

开放性创新实践软件项目 2023.04.23 结项总结

1 项目搭建过程

模仿老师思路，自我独立实现，好友头脑风暴

1 需求分析，详细设计以及环境准备

选题后开始分析文章管理系统的最终功能需求并构思可能方案，最初设计的生物文献管理系统虽然没有实现，但是为后来的电影艺术文章管理系统提供了更多可行的思路与方案，整体的需求分析与详细设计实际上是跟随开发过程一同进展的。
此外按照老师的视频笔记以及网络教程安装配置数据库，前后端开发环境并尝试使用 Git 进行项目版本管理，之后随着开发安装配置 ElasticSearch。

2 基于 SpringBoot 的 Java 后端

将老师的后端代码 Fork 后 Git Clone 到本地，通过 Maven 安装配置各种依赖项。
